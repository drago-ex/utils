## Requirements
- PHP 8.0 or higher
- composer

## Installation
```
composer require drago-ex/utils
```

## Returns items as array
```php
$data->toArray();
```

## Returns items as array with converted keys to uppercase
```php
$data->toArrayUpper();
```
